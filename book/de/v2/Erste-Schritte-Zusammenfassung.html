Zusammenfassung

Sie sollten jetzt ein grundlegendes Verständnis dafür haben, was Git ist und wie es sich von anderen zentralisierten Versionsverwaltungssystemen unterscheidet, die Sie evtl. bisher eingesetzt haben. Sie sollten nun auch eine funktionierende Version von Git auf Ihrem System haben, die mit Ihrer persönlichen Identität eingerichtet ist. Es ist jetzt an der Zeit, einige Grundlagen von Git zu lernen.
