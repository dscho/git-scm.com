Ανακεφαλαίωση

Έχουμε καλύψει τους περισσότερους από τους σημαντικότερους τρόπους με τους οποίους μπορούμε να εξατομικεύσουμε τον πελάτη και τον διακομιστή Git ώστε να ταιριάζουν καλύτερα στη ροή εργασίας και τα έργα μας. Έχουμε μάθει για όλα τα είδη των ρυθμίσεων διαμόρφωσης, τα γνωρίσματα που βασίζονται σε αρχεία και τα άγκιστρα συμβάντων και έχουμε δημιουργήσει ένα παράδειγμα διακομιστή με επιβολή πολιτική. Πλέον είμαστε σε θέση να εξατομικεύσουμε το Git σε σχεδόν οποιαδήποτε ροή εργασίας.
