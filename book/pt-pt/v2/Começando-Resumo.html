Resumo

Tu deves ter uma compreensão básica do que é o Git e como é diferente de qualquer sistema de controle de versão centralizado que tu tenhas usado anteriormente. Também deves agora ter uma versão funcional do Git no teu sistema configurado com a tua identidade pessoal. Agora é hora de aprenderes alguns princípios básicos do Git.
