Хулоса

Ушбу вақтга келиб сиз Git устида бажариладиган асосий барча маҳаллий амалларни бажаришни биласиз: омборни яратиш ва клонлаштириш, унга ўзгартириш киритиш, индекслаш ва ўзгаришларни фиксирлаш шунингдек омбордаги барча ўзгаришларнинг тарихини кўриш. Давом этиб биз Git нинг жуда кучли имконияти ҳисобланмиш – тармоқланиш модели ҳақида гаплашамиз.
