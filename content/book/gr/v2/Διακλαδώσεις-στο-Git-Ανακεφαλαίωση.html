Ανακεφαλαίωση

Καλύψαμε τα βασικά της δημιουργίας και συγχώνευσης κλάδων στο Git. Πλέον αισθανόμαστε αυτοπεποίθηση να δημιουργούμε κλάδους, να μεταβαίνουμε σε αυτούς και να συγχωνεύουμε τοπικούς κλάδους. Ακόμα μπορούμε να μοιραζόμαστε τους κλάδους μας ωθώντας τους σε έναν κοινόχρηστο διακομιστή, να δουλεύουμε με άλλους σε κοινούς κλάδους και να αλλάζουμε τη βάση των κλάδων μας προτού τους μοιραστούμε με άλλους. Στη συνέχεια θα εξετάσουμε τι χρειάζεται ώστε να λειτουργήσουμε έναν δικό μας διακομιστή που θα φιλοξενεί ένα αποθετήριο Git.
