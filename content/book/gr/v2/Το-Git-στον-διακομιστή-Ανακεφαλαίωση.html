Ανακεφαλαίωση

Έχουμε αρκετές επιλογές για το πώς να δημιουργήσουμε και λειτουργήσουμε ένα απομακρυσμένο αποθετήριο Git έτσι ώστε να μπορούμε να συνεργαζόμαστε με άλλους ή να μοιραζόμστε την εργασία μας.

Η λειτουργία του δικού μας διακομιστή μάς δίνει πολύ έλεγχο και μας επιτρέπει να τρέχουμε τον διακομιστή εντός του τείχους προστασίας μας αλλά ένας τέτοιος διακομιστής γενικά απαιτεί αρκετό χρόνο για να εγκατασταθεί και να συντηρείται. Αν τοποθετήσουμε τα δεδομένα μας σε έναν φιλοξενούμενο διακομιστή, είναι εύκολο να εγκατασταθεί και να συντηρείται· ωστόσο, πρέπει να είμαστε σε θέση να διατηρήσουμε τον κώδικά μας σε διακομιστές τρίτων και ορισμένοι οργανισμοί δεν επιτρέπουν κάτι τέτοιο.

Θα πρέπει να είναι αρκετά απλό να προσδιορίσουμε ποια λύση ή συνδυασμός λύσεων είναι κατάλληλη για εμάς και τον οργανισμό μας.
