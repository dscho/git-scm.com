НАЗВАНИЕ

git-init-db - Создание пустого репозитория Git

ОБЗОР

git init-db [-q | --quiet] [--bare] [--template=<template-directory>] [--separate-git-dir <git-dir>] [--shared[=<permissions>]]

ОПИСАНИЕ

Эта команда является синонимом git-init[1]. Смотрите документацию оной.

GIT

Является частью пакета git[1]
